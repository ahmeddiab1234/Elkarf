NYC Trip Duration Project

  • Predicts trip durations in New York using 10 features and 1M training samples.
  • Best model achieved with XGBoost Regressor — F1-score: 76% (Val) & 72% (Test).
  • Frontend built with Streamlit and backend powered by FastAPI.

Credit Card Fraud Detection

  • Classifies fraudulent transactions using 31 features and 56,960 examples (with 28 anonymized).
  • Handled high class imbalance (0.172% positive ratio) focusing on F1-score and recall.
  • Achieved top performance with Random Forest: Val_F1 = 89%, Test_F1 = 80%.

Mini_Rag Application

  • Built a minimal, educational RAG (Retrieval-Augmented Generation) question answering system.
  • Integrated LLMs, vector databases, and backend tools like FastAPI, Docker, Celery, and PostgreSQL.
  • Step-by-step guidance included for learning purposes.

Car Price Prediction

  • Predicts car prices based on multiple features including ID, Price, Levy, Manufacturer, Model, Production year, Category, Leather interior, Fuel type, Engine volume, Mileage, Cylinders, Gear box type, Drive wheels, Doors, Wheel, Color, Airbags, and Random_notes.
  • Implemented using regression models to estimate accurate pricing.
